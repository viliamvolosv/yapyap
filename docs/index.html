YapYap Messenger

A decentralized, peer-to-peer messenger node and CLI library with end-to-end encryption, offline delivery, and no UI.

Get Started View on GitHub

Key Features

Built for reliability, security, and seamless integration

End-to-End Encryption

Secure messaging using Noise XX/IK and Ed25519 cryptography for all communications.

Offline Delivery

Store-and-forward messaging with SQLite persistence and deduplication for reliable delivery.

ACK-Driven Reliability

Message acknowledgment system with automatic retry and exponential backoff for guaranteed delivery.

Decentralized P2P

Bootstrap + DHT discovery with autonat/relay fallbacks for reliable peer connectivity.

CLI & Library

Run as standalone CLI or embed as a library in your applications with full control.

SQLite Persistence

Robust database schema with message queue, processed messages, and replica tables.

Getting Started

Start using YapYap in minutes

1

Install Bun

Ensure you have Bun 1.3+ installed

bun --version
2

Install Dependencies

Clone the repo and install packages

bun install
3

Build the Project

Compile the CLI and library

bun run build:cli
4

Run YapYap

Start the node and start messaging

./dist/cli.js
